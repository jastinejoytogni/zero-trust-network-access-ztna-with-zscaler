# Zero Trust Network Access (ZTNA) with Zscaler

This capstone project focuses on designing and prototyping a Zero Trust Network Access (ZTNA) solution using Zscaler Private Access (ZPA) within an education technology context. Teams will critically assess the limitations of traditional perimeter-based and VPN-centric security models, which often lead to broad network access and increased attack surfaces. The project requires students to apply foundational cybersecurity principles, including vulnerability assessment, least privilege access, and micro-segmentation, to secure distributed environments.
